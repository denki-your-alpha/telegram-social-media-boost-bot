# Telegram Social Media Boost Bot 🚀

A feature-rich Telegram bot for social media account boosting with payment processing, referral system, and admin panel.

## Features ✨

- 🎯 **Boost Services** - Instagram, TikTok, YouTube, Twitter
- 💰 **Payment System** - Multiple payment accounts with screenshot verification
- 👥 **Referral Program** - 10% commission per referral
- ⚙️ **Admin Panel** - User management and payment approval
- 🔐 **Channel Verification** - Required channel subscriptions
- 📊 **User Dashboard** - Balance, orders, referrals tracking

## Prerequisites 📋

- Node.js v14 or higher
- npm or yarn
- Telegram Bot Token (from @BotFather)
